#运算符、表达式和语句

## 左值、右值、运算符

| 术语 | 含义 |
| --- | --- |
| 数据对象 | 用于储存值的数据存储区域，也称对象 |
| 左值 | 用于标识特定数据对象的名称或表达式；可修改的左值用于标识可修改的对象，又称对象定位值。 |
| 右值 | 能赋值给可修改左值的量，本身不是左值 |
| 运算符 |  |

## 运算符优先级

| 优先级 | 运算符 | 结合方向 | 类型 |
| --- | --- | --- | --- |
| 1 | () ; [] ; -> ; . | -> | 括号类 |
| 2 | ! ; ~ ; ++ ; -- ; - ; (transform) ; * ; & ; sizeof | <- | 单目 |
| 3 | * ; / ; % | ->| 乘除类 |
| 4 | + ; - | -> | 加减 |
| 5 | << ; >> | -> | 左右移动 |
| 6 | （大小关系等）| -> | 关系 |
| 7 | （相等关系等运算符）| -> | 相等 |
随后优先级依次降低为：  

按位: 与 >异或 >或 > 逻辑: 与 > 或  

随后结合方向变为`从右至左`: `? :` > 赋值运算符  > 逗号
